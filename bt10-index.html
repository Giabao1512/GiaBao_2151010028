Giới thiệu môn học

Môn học Ứng dụng web trang bị cho sinh viên các kiến thức cơ bản về ngôn ngữ HTML, Javácript, CSS, jQuery; cách lập trình xử lý trang web động phía trình duyệt, sử dụng một số phầm mềm thiết kế Web phổ biến
